## CSS:
Mô hình RGB (Red-Green-Blue): là một hệ thống màu sắc dựa trên 3 màu cơ bản (Đỏ, Xanh lá, Xanh dương)
Nguyên lý hoạt động:
Cộng màu: khi chúng kết hợp thì tạo ra màu sáng hơn
Giá trị từ 0 - 255: 0 (màu tối nhất) - 255 (màu sáng nhất)
Các màu cơ bản:
+ rgb(255, 0, 0)
+ rgb(0, 255, 0)
+ rgb(0, 0, 255)
Các màu thứ cấp (secondary): 
+ rgb(255, 255, 0)
+ rgb(0, 255, 255)
+ rgb(255, 0, 255)
Các màu bậc 3 (tertiary):
+ rgb(255, 125, 0)
+ rgb(0, 255, 125)
+ rgb(255, 0, 255)
